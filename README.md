# ShellColors

### Install in TermUX ( Android )

```bash
pkg install git -y 
PWDx=$PWD
cd ~
git clone https://github.com/ShivaShirsath/ShellColors.git
bash ~/SheolColors/install
cd $PWDx
```
### Generate Code 
```bash
getColorCode
```

### Use code
```json
echo -e "[ long press & paste Color Code ]Your Text !"
# Or
printf "[ Long press & paste Color Code ]Your Text !"
```
